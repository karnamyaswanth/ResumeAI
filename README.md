# 📄 ResumeAI – AI Resume Analyzer

An intelligent AI-powered Resume Analyzer built using Python and Streamlit that evaluates resumes, extracts skills, calculates ATS scores, recommends suitable job roles, and generates personalized resume improvement suggestions.

---

## 📌 Project Overview

ResumeAI helps users improve their resumes using automated analysis and smart recommendations.

Users can:

* Upload resumes (PDF/DOCX)
* Extract resume text automatically
* Detect technical and professional skills
* Calculate ATS (Applicant Tracking System) score
* Receive job recommendations
* Identify missing skills
* Get AI-generated improvement suggestions

## ⚙️ Installation

Clone Repository

```bash
git clone https://github.com/karnamyaswanth/ResumeAI.git
```

Open Folder

```bash
cd ResumeAI
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Application

```bash
streamlit run app.py
```
